Puesto que en GitHub el tamaño máximo de los archivos en un repositorio no debe exceder los 100MB, no se puede adjuntar la base de datos para su descarga directa de GitHub. No obstante, adjuntamos la liga a la página de datos abiertos del gobierno de méxico en el cual se puede hacer la descarga directa de la base de datos, llamada "Defunciones" en nuestro código.
